An AI-powered interview preparation platform that helps users practice, learn, and improve their interview skills with real-time feedback and smart features.


## ✨ Features

- 🔐 Authentication (Google OAuth).
- 🤖 AI-powered interview questions & responses.
- 💳 Payment Integration (Razorpay).
- 📊 User Dashboard & Progress Tracking.
- ⚡ Fast and responsive UI.
- 🔒 Secure API with protected routes.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### Other Tools
- Razorpay (Payments).
- Firebase (Auth).
- Vercel (Frontend Deployment).
- Render (Backend Deployment).
